Terminal Based Monoply

This is a java project that runs a virtual game of the two player Monoply. Rules are standard to the orignal. Some more advanced game features are missing at the moment. If being run on windows you may need to input a character before pressing enter to regester a move.

Latest Release:
I have been using eclipse as my IDE for this project –not an online editor– and this project has not been pushed to Github. You can find and download the latest post of the game here.
